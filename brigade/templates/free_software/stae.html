What is Stae?

Stae is a universal tool for data management and collaboration. Cites can integrate data from any source, map different data types, and create real-time data feeds.

Brigade projects can use Stae to publish community data or to consume it. Many common types of data (311 data, traffic data, 911 calls, etc.) are standardized across many jurisdictions, which enables more scalable projects.

Read more about Stae on the Code for America Discourse

What do you get?

Currently, Stae is open to the public and is free.

How do you get access?

Simply click the "Sign Up" button on the Stae product homepage.

If you find Stae useful in a project, let us know!
