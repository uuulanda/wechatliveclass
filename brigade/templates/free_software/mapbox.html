Mapbox

What is Mapbox?

Mapbox helps you build custom maps and data visualizations easily. They offer many products, including Maps, Geocoding, and Mapbox Studio (for custom maps).

Mapbox has generously offered brigades access to these products to get products off the ground!

What do you get?

To start with, we'll grant you an API token which can be used with their Maps or Geocoding service.

This API token will support unlimited usage, though you should be respectful of Mapbox rate limits.

How do you get access?

Email inkinds@codeforamerica.org with "Mapbox Request" in the subject line and we will send you an access token. We'd also love to learn a little bit about the project you're hoping to use Mapbox for!

What are the expectations?

Mapbox has generously donated their services to get your brigade project off the ground. We will ask that if your project ends up being adopted by a city, or begins to earn money, to migrate off of Code for America's mapbox account onto your own. Email community@mapbox.com if you find yourself in this situation.
