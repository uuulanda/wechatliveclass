About the Code for America Network Team

The Code for America Network Team supports the nationwide Brigade Network, organizes the Code for America Community Fellowship, and leads Code for America’s volunteer mobilization. To do this, the team works closely with Brigade leaders, the National Advisory Council (NAC), and national partner organizations to develop and execute a program strategy to build a nationwide movement.

Meredith Horowski

As the Senior Director of the Code for America Network, Meredith establishes and leads the strategy for the Brigade Network, Community Fellowship, and volunteer mobilization in collaboration with Brigade members, our National Advisory Council (NAC), fellows, alumni, partners, and Code for America stakeholders.
